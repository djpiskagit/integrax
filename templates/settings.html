Прокси SOCKS5 / MTProto

Нужен если хостинг блокирует прямое подключение к Telegram

❌
Хостинг блокирует MTProto — прямое подключение к Telegram невозможно
Порты 443 / 80 / 5222 Telegram недоступны напрямую с вашего хостинга.
Настройте SOCKS5 или MTProxy ниже — тогда все запросы пойдут через прокси.
🌐 Webshare.io — бесплатный SOCKS5 ✈ @MTProxybot — MTProto
▸ Тип подключения
Авторизация (опционально)
Из ссылки: tg://proxy?server=…&port=…&secret=ВОТ_ЭТО
▸ Как получить SOCKS5 бесплатно — Webshare.io
1. Зарегистрируйтесь на webshare.io — бесплатный план: 10 прокси, карта не нужна
2. Dashboard → Proxy List → выберите любой прокси
3. Скопируйте Host, Port, Username, Password
4. Вставьте в поля выше, нажмите 💾 Сохранить, затем ⚡ Тест
▸ Как получить MTProxy — @MTProxybot
1. Откройте @MTProxybot в Telegram и отправьте /getproxy
2. Из ссылки вида tg://proxy?server=X&port=Y&secret=Z
   скопируйте: server → Host, port → Port, secret → Secret
3. Вставьте выше, нажмите 💾 Сохранить, затем ⚡ Тест
